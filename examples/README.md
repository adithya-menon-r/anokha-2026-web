# Examples — Anokha 2025 Frontend

This folder contains reference code patterns for developers and AI tools.

Use these examples when building new features to match project style.

## Contents

- `/services/` — API services using `apiGet<T>` / `apiPost<T>`
- `/hooks/` — Data hooks using TanStack Query object form
- `/components/` — Dumb UI components (pure UI, typed props)
- `/features/` — Smart components (state + data logic)
- `/stores/` — Zustand stores (with or without persist)

## Usage Notes

- Do not edit example files — they are reference only.
- Match the style and architecture shown here when writing code.
- Copilot Chat and Copilot Reviewer also use these examples for context.

## When in doubt

If you are unsure how to write any service, hook, store, or component —  
→ Always check here first before writing your own pattern.
